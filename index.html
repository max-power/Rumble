Rumble by max-power

Rumble is a jQuery/Zepto plugin that rumbles, vibrates, shakes, and rotates any element you choose. It's great to use as a hover effect or a way to direct attention to an element. You can control the X, Y, and rotation range, the speed, and event trigger for the rumble. It is based on jRumble by Jack Rugile (http://jackrugile.com).
