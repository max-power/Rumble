The Plugin

Rumble is a jQuery/Zepto plugin that rumbles, vibrates, shakes, and rotates any element you choose. It's great to use as a hover effect or a way to direct attention to an element. You can control the X, Y, and rotation range, the speed, and event trigger for the rumble.

Note: Rumble will work only in Webkit based browsers.

Usage

Include jQuery or Zepto and the Rumble plugin.

<script type="text/javascript" src="javascripts/zepto.min.js"></script>
<!--script src="http://ajax.googleapis.com/ajax/libs/jquery/1.5/jquery.min.js">
<script type="text/javascript" src="javascripts/rumble.min.js"></script>

Reference Rumble on a Selector

You can do this in a script tag within your HTML or through an external Javascript file.

$(document).ready(function(){
  $('#my-rumble-div').rumble();
});

You can pass some options:

$(document).ready(function(){
  $('h1,h2,li').rumble({
    x: 10,      // X offset
    y: 5,       // Y offset
    a: 20,      // Rotation angle
    speed: 30,  // animation speed in ms
    on: 'hold'  // when the rumble will start: 'hover', 'hold', 'toggle', 'constant'
  });
});
